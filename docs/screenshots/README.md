# Screenshots Directory

This directory contains screenshots of the SplitRent application.

## How to Add Screenshots

1. Take high-quality screenshots (recommended resolution: 1920x1080 or higher)
2. Save them in this directory with descriptive names
3. Update the paths in README.md to point to your actual files

## Recommended Screenshots

- `landing-page.png` - Homepage/landing page
- `dashboard.png` - Main dashboard view
- `wallet-connect.png` - Wallet connection modal
- `payment-form.png` - Payment form interface
- `transaction-history.png` - Transaction history list
- `dark-mode.png` - Dark mode theme example
- `light-mode.png` - Light mode theme example
- `mobile-view.png` - Mobile responsive view
- `escrow-detail.png` - Escrow detail page (when implemented)
- `onboarding.png` - Onboarding checklist

## Screenshot Guidelines

### Quality Standards

- Use retina/high-DPI display if possible
- Ensure all text is readable
- Remove browser extensions that might clutter the UI
- Use a clean browser window (hide bookmarks bar if distracting)
- Consider using a consistent background color

### Tools

- **macOS**: Cmd+Shift+4 for selection, Cmd+Shift+5 for more options
- **Windows**: Win+Shift+S for snipping tool
- **Linux**: Depends on distribution (e.g., Flameshot, Shutter)
- **Browser Extensions**: Full Page Screen Capture, Nimbus Screenshot

### Editing (Optional)

- Add annotations with tools like Skitch, Snagit, or GIMP
- Highlight important features with arrows or boxes
- Blur sensitive information (addresses, amounts)
