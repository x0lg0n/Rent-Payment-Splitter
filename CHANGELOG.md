All notable changes to SplitRent will be documented in this file.

### Added

- Initial release
- Basic wallet connection functionality
- Payment sending capability
- Transaction history display
- Soroban escrow contract foundation
- Next.js frontend with App Router
- TypeScript type safety throughout the codebase
- Tailwind CSS styling with shadcn/ui components
- Vitest testing setup
- ESLint and TypeScript type checking
- GitHub Actions CI/CD pipeline
- Comprehensive open source documentation
